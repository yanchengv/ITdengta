软件开发

PHP

PHP是一种通用开源脚本语言，语法吸收了C语言、JAVA、Perl的特点，入门门槛较低，易于学习，开发效率高，使用广泛主要是用于web开发领域，目前国内80%以上的网站（包括淘宝、百度、新浪等）都使用了PHP技术。

HTML5

HTML5开发技术是一种多语言综合开发技术，每种语言在HTML5开发过程中各行其责。使用HTML5可以开发桌面平台和移动平台的响应式布局，使用HTML5可以开发跨平台的移动端APP

JAVA

JAVA语言是一种高级编程语言，以其独有的开放性、跨平台性以和面向网络的交互性，以其安全性易用性和开发周期短等特点，迅速从最初的编程语言发展成全球第一大软件开发平...

Android

Android,主要使用于移动设备，如智能手机和平板电脑，由Google公司和开放手机联盟领导及开发。可以开发andoid系统上的游戏、应用等，目前市场对android的需求很大

Ruby on Rails

(官方简称为Rails,ROR非官方简称)是一个使用Ruby语言写的开源Web应用框架， 它是严格按照MVC结构开发的。它努力使自身保持简单，来使实际的应用开发时的代码更少，使用最少的配置。Rails的设计原则包括"不做重复的事"和"惯例优于设置"。

雅思

雅思考试（IELTS），外文名International English Language Testing System，由剑桥大学考试委员会外语考试部、英国文化协会及IDP教育集团共同管理，是一种针英语能力，为打算到使用英语的国家学习、工作或定居的人们设置的英语水平考试。

托福

托福是由美国教育测验服务社（ETS）举办的英语能力考试，全名为“检定非英语为母语者的英语能力考试”，中文音译为“托福”。

英语早读

早读字面意思是早晨读书，可以默读或大声朗读。但最好大声朗读，早晨读书有益于记忆，读的可以是课本、笔记或是报纸、杂志。大多学校把早读当做早晨学习任务，通常需要开口朗读并且要求整齐阅读（便于管理因素）。

考研政治、英语

硕士研究生教育按照培养目标的不同，分为学术型研究生和专业学位研究生。学术型研究生的培养目标是高层次学术研究型专门人才，专业学位研究生的培养目标是具有扎实理论基础，并适应特定行业或职业工作需要的应用型高层次专门人才。
